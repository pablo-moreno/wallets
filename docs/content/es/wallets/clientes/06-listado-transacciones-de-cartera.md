<alert type="warning">

El usuario debe estar logueado y ser de tipo `cliente (1)`.

</alert>


### Petición

<code-block label="Bash" active>

```
GET /api/v1/wallets/customers/wallets/<uuid>/transactions
```

</code-block>

### Respuesta

> `200 OK`
